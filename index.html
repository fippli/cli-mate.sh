CLIMATE

climate|

Climate is a minimalistic script manager written i bash.

The purpose of climate is to make it easy for you to install and run scripts that are published in a git repository, or makes your scripts easy for others to install!

Installation

Detailed instructions are maintained in the code repository readme. To quickly install you can run the command below: 

          bash <(curl -s https://raw.githubusercontent.com/fippli/climate/master/easy_install)

Make sure that you export ${HOME}/.climate/bin to your path.

Basic usage

The main features of climate are install, remove, and update e.g.

Install a package

          climate install <git-repository-url>

This will clone the code from github and put it in ${HOME}/.climate/bin/<script-name>. Where <script-name> is the name configured in the .climate file in the package repository.

Update a package

          climate update <script-name>

This will clone the recent code from the repository connected to the <script-name> and replace the previous version.

Remove a package

          climate remove <script-name>

Will remove all the files associsated with the <script-name>.
